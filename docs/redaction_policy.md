# Redaction Policy

This repository may keep the public project name `Believe in Me`.

It must not contain:

- real unpublished rules
- real card names, card text, or asset filenames
- real internal reports, traces, or private test notes
- private project-management links or dashboards
- local credential paths or secrets
- production data files copied from another repository

The demo model should stay generic. If a detail is only useful because it
matches the private board game, it does not belong here.
